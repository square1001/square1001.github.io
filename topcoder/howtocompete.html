Topcoderに参加しよう by square1001

Topcoderは、2001年から長く続く歴史のある国際的なプログラミングコンテストサイトで、さらに今も世界有数のプログラミングコンテストサイトとなっています。Topcoderでは、SRMやMarathon Matchという競技プログラミングのコンテストのみならず、いろいろなコンテストを開催しています。

まず言っておきたいのは、Topcoderは競技プログラミングだけのサイトではなく、さらにDesign、Development、Data Scienceの分野があり、ウェブサイトが必然的に複雑になってしまうということです。なので、今TopcoderのSRMに参加するシンプルな方法や知っておくべきことをここに書きます。

これは2018年1月25日時点のものです。数年たったら変化しているかもしれないことにご注意ください。

1. ユーザー登録

Topcoderのユーザー登録の方法を今から説明します。まず、Topcoderのウェブサイトは https://www.topcoder.com/ です。このページに移動すると、次のような画面が出てきます。

ユーザー登録をするためには、「I want to join Topcoder」をクリックしてください。その後「Become a member」をクリックすると、次のようなページが出てきます。

このページは、https://accounts.topcoder.com/member/registration という場所です。ここでは、ユーザー登録をするフォームがあり、次の6つを入力しなければなりません。

  • First Name - 名前 (名) を英語表記で
  • Last Name - 苗字 (姓) を英語表記で
  • Country - 国籍 (これを読んでいるほとんどの人は「Japan」と打てばOK)
  • Username - ユーザー名 (square1001 とか)
  • Enter your email - Eメールアドレス
  • Create Password - パスワード
その後、入力したメールアドレスに確認用メールが届くと思うのでこれで認証をします。

これでユーザー登録ができました。

2. Topcoder SRM のベース - Topcoder Arena の利用

Topcoder の SRM に参加したり、過去の問題を解いたりするときに、Topcoderの競技プログラミングの競技場 - 「Topcoder Arena」が必要になってきます。最近ウェブサイト上で動く競技場 Web Arena もありますが、これはまだ開発段階なので Applet Arena の方を使うことをお勧めします。ここで、Applet Arena の使用方法を説明します。

まず、Applet Arena を使うためには、Java の環境が必要になってくるので、Java の環境がない人は https://java.com/en/download/win10.jsp などから Java 環境をダウンロードしインストールしましょう。

次に、Applet Arena のファイルをダウンロードします。まず、ログインした直後の画面 (Dashboard) は次のようになっています。

ここで「LEARN --> COMPETITIVE PROGRAMMING」の部分をクリックすると、次のような画面が出てきます。

このページで「LAUNCH APPLET ARENA」をクリックすると、Applet Arena がダウンロードされます。ここからはファイルは手元で開くことができます。

次に説明するのは、「セキュリティ設定」についてです。ファイルを開くときに、「セキュリティ設定によってブロックされたアプリケーション」というエラーが発生する場合があります。その場合、スタートメニューから「Java」-->「Javaの構成」をクリックします。すると、「Java コントロールパネル」が開き、「セキュリティ」をクリックしましょう。最後に、「例外サイト・リスト」に「http://www.topcoder.com」を追加します。すると Applet Arena が開けるようになります。

Applet Arena を開くと、次のような画面が出てきます。

「Username」のところにユーザー名、「Password」のところにTopcoderのパスワードを入れましょう。そして、「GO」を押すとログインが完了します (まれに数分くらいかかることがあります)

この画面がTopcoder Applet Arenaのロビーです。この競技場でSRMに参加したりSRMの過去問を解いたりできます。
